# 全国大学生数学竞赛 LaTex模板

该项目是高教杯全国大学生数学竞赛的latex模板，延续了一些他人模板的设置


## 建议环境

vscode + latexworkshop插件 + WSL环境
好用的插件：HyperSnips, WSL, Vim, GitHub Copilot, Ultra Math Preview
